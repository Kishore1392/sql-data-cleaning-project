# SQL Data Cleaning Project

## Business Problem
Raw business data often contains duplicates, missing values, and inconsistent formats.
This project focuses on cleaning messy data to make it analysis-ready.

## Dataset Description
The dataset contains customer and sales-related records with common data quality issues.

## Tools Used
- SQL
- MySQL / PostgreSQL

## Steps Performed
- Removed duplicate records
- Handled NULL values
- Standardized text fields
- Corrected date formats
- Validated numeric ranges

## Output
A clean, structured table ready for analysis and reporting.

## How to Run
1. Import the dataset into a SQL database
2. Execute the queries in order from the SQL script
